与热爱同行

发表于 2021-01-03   |     |  

桌上的台历已经翻完最后一页，合上这充满诸多集体回忆的一年，所有辛与不幸，都封存成历史。有些人永远的停留其中，与所有不平凡一起，烙在了从中经过的人的生命里。

年初武汉封城的消息传出之后，人心惶惶，没人想着怎么过年。我担心疫情蔓延之后四处封城，在老家待了两天就搭上回广州的火车。从小没有亲身经历过疫情，那些日子里心都是悬着的，像走高空栈道般战战兢兢。空荡的街道，大喇叭里循环播放着防疫宣传语，一遍遍拨弄着人紧绷的神经。

大年初二晚，我辗转反侧无法入眠，半夜打开手机，先后看到两条消息：国务院宣布因疫情影响，春节假期延长三天；科比在直升机撞山事件中离世……我不相信是他，到处寻找哪怕一条质疑该消息真实性的报道，但得到的却是更令人心碎的细节——他最具篮球天赋的二女儿 GiGi 也在出事的直升机上……那天早上，我从悲伤的梦境中醒来，泪水却没能停住……接下来的两天，我陷入了虚无，不真实感包围着我，每一次想到这个噩耗都会让情绪决堤……直到一年后的现在，看回乔丹对他的追悼讲话，我的情绪依然不能自已。

篮球是从初中开始陪伴我到现在的爱，而科比是我们这一代人的「篮球之神」。中学饭堂的电视机前，一群人端着饭兜仰望你的比赛；与同学打赌湖人 F4 阵容必定夺冠却最终折戟；大学暑假，科比「中国行」来了我们学校，体育馆被围得水泄不通，大家只为一睹真容；工作后，又与年轻同事争论你和詹姆斯，谁是联盟第一；你退役前最后一战狂砍 60 分取胜，我激动得欢呼惊叹……

拼到跟腱断裂，依旧眼神坚毅命中罚篮的画面；「凌晨四点洛杉矶」训练场中的身影；科比对比赛的偏执，对篮球的狂热——这些早已经化作图腾，融入我们的生命。我不知道多少人在出手一个投篮后心中会默念「科比」，但科比对热爱的篮球倾尽所有的毅力，却真实激励着我——只有坚定地投入到热爱的事物中，才能改变你的人生。

2016 年初，我辞去了毕业后的第一份工作，随后加入腾讯，但又很快离开。办理完腾讯离职手续的那个下午，回到家中，我看着窗外，眼泪悄然落下。腾讯是我当时最想加入的公司，即便如此，面对不够热爱的领域，自我定位和价值感的缺失，依然让我丧失激情、陷入迷惘。那么我到底热爱什么？带着这个疑问，我进入了人生第一个「间隔年」。

寻找人生方向，要「向外看、往内寻」。我开始广泛涉猎、自由探索——观看《Cosmos》纪录片，阅读《人类简史》《自私的基因》……通过对宇宙、人类、社会以及自身的深刻思考，重塑基础认知，发现内在动机，以作为一切行动的基础。

某一天，我整理大学时收藏的演讲视频时，偶然看到了 Randy Pausch 教授的《最后一课》。我突然发现，当我以「创造打动人心的互动娱乐作品」作为使命时，我之前所有的经历、所有的思考、所有的渴望，都神奇地串联起来了！我生命中的每一个「点」，自然地串联成了一条「线」。那一刻我就像一颗种子破土而出、伸展枝芽。拥有完整的梦想让我内心之火第一次熊熊燃起，包围我的迷雾瞬间消散，生命之光落在我身上，温暖明亮又充满力量！

明确了人生使命和内在动机之后，我紧接着探索如何构建思维与行动体系，打造认知的「操作系统」。《赖声川的创意学》中的「创意金字塔」构成了这个「操作系统」的架构，而《穷查理宝典》中的「普世智慧」则构成了思维与行动的准则。

此外，践行了多年的「时间管理」让我得以坚实地踏出每一步，而不是做「思想上的巨人，行动上的矮子」。受到《奇特的一生》启发，我又尝试着更加细致地记录每一处时间开销，以作出更合理的时间规划。如今，在生活中随手记录时间开销，于我已如呼吸般自然。自我认知偏差总是难免，我们有时会高估自己对某项事物的喜爱（比如阅读学习），有时会低估自己完成某项任务所需付出的努力，但时间开销数据不会说谎，时常复盘时间支出，让我们更加客观的了解自己。

2017 上半年，我开始自学游戏开发知识，以制作游戏作为追逐梦想的起点。后来，加入了一个初创的独立游戏团队，本以为征程就此启航，没想到三个月后团队就因资金链断裂而解散，被拖欠的一个月工资到现在还没有着落……后来几经辗转，在年底又加入了一个处于上升期的小团队，并一直工作至今。三年来，我一头扎进工作里，把百分之百的时间精力投入其中，只为了追随内心的呼唤。

拥有了明确的内在动机，行动上自然无需依赖于外在奖赏，自我成长即是最大回报。但「人是社会的人」，在追逐自我实现的同时，还要满足生存的基本需求。由于我是跨行转职，一切都得从零开始。而为了避免家人担心，至始至终我都没向家人透露自己职业的巨大转变以及空窗一年的事实，所以家人日常的赡养费用还是照常供应。入不敷出，有两年时间里我的经济状况都非常紧张，甚至一度快交不上房租……

全力奔跑时，容易忽略身边人的感受。这几年也经历了几次情感危机，逐渐学会如何在亲密关系中倾听彼此的诉求，尊重他人的想法与抉择，而不是固执己见甚至强加于人。旅途的意义，不在于终点，而在于旅途之中——这是我游玩《Journey》的感悟，也是诗歌《伊萨基》所阐述的道理。

几个月前无意中发现我的 GitHub 首页上多了一个 Arctic Code Vault Contributor 勋章，原来是我有一个库被加入了 GitHub Archive Program。该归档计划是将那些活跃的公共源码库保存在超过 1000 年寿命的胶卷中，然后存储在北极的永久冻土区中，以确保千年后的人类依然能够受益于这些知识……那一刻，我觉得所有的付出都是值得的。投身于热爱，而其馈赠就是热爱本身。

Mamba Forever.

蔡子聪
2021.1.3
